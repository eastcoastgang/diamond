Installation

Diamond installation is meant to be very easy. We use only a few common external libraries. You can find our setup template below containing all the required imports for a basic HTML project.

Components

Available components are listed below along with code samples.



Text Buttons

Use the <button> element with class .d-btn, .text or .contained, and .primary or other colors. For example, a contained danger button would simply contain the following classes: d-btn contained danger.



Contained Buttons



Cards

Use the <div> element with class .d-card. You can specify shadow intensity with .light or .heavy

Here is a light card with classes: d-card light.

Here is a default card with classes: d-card.

Here is a heavy card with classes: d-card heavy.

Lists

Use the <ul> element with class .d-list.
